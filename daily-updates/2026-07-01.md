## 💡 Lessons Learned (added to repo-creator + repo-pusher sub-skills)

### 1. `gh repo create --source` corrupts local git config
- **Symptom:** `git push` says `'origin' does not appear to be a git repository`
- **Fix:** `git remote add origin <url>` and `git push -u origin main` again
- **Lesson:** Use `gh repo create <name> --confirm` (without `--source`), then push manually
- Last verified: 2026-07-01

### 2. Token-bucket decorator without `timeout` waits forever
- `acquire(timeout=None)` blocks indefinitely when over-limit
- Decorator without timeout makes the wrapped function unreachable
- **Fix:** validate `timeout` at decoration time; add `timeout=0.0` fast-path
- Same lesson applied to FixedWindow and SlidingWindow
- Last verified: 2026-07-01

### 3. `asyncio.Semaphore` + Task tracking needs care
- AsyncPool tracks in-flight tasks for `join()`
- Must create the task **inside** the semaphore acquire
- **Correct order:** `async with sem: task = create_task(coro); await task`
- Reverse order leaks tasks past the semaphore
- Last verified: 2026-07-01
