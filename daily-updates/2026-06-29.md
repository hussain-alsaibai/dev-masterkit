## Lessons Learned

### GitHub Push Protection Traps
- **Problem:** A pre-existing PAT leaked in old commits blocks ALL pushes (even clean ones) to the repo
- **Workaround:** `gh repo delete --yes` then `gh repo create --push` to start fresh
- **Lesson:** Rotate compromised PATs and rewrite history with `git filter-repo` if the repo must be preserved
- **Last verified:** 2026-06-29

### Submodule `.git` Reset Hazard
- **Problem:** Local submodule `.git` is shared with parent repo's working tree
- Running `git reset --hard origin/main` from inside a submodule wipes the `.git` directory
- **Fix:** Never run `git reset --hard` inside a submodule
- Use `git checkout` or operate from the parent repo
- **Last verified:** 2026-06-29

### Stdlib Aliasing Pattern
- **Problem:** `Decimal(value)` resolved to `Decimal_` (our alias) instead of stdlib `Decimal`
- Caused silent type coercion bugs in tiny-validator
- **Fix:** Import stdlib as `_Decimal` and alias at the end:
  ```python
  from decimal import Decimal as _Decimal
  Decimal = _Decimal  # public alias
  ```
- **Last verified:** 2026-06-29
