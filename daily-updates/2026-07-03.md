## 💡 Lessons Learned (added to zero-dep-pattern + repo-creator skills)

### 1. `_process_lock` class shadows method name → infinite recursion
- Defined `class _process_lock` inside `Queue` and a method `_process_lock()`.
- Inside the method body, `return Queue._process_lock(self)` resolved to the *method* (not the class) → infinite recursion.
- **Fix:** capitalize the inner class (`_ProcessLock`) and keep the method lowercase. Or use a different attribute name.
- **Promotion:** added to `zero-dep-pattern` skill — "Don't let helpers share names with their enclosing scope".
- Last verified: 2026-07-03

### 2. `fcntl.flock` is unreliable on tmpfs in Docker
- The test `test_purge_clears_all` hung indefinitely.
- Root cause: `fcntl.flock` on a file in `tmpfs` blocks even within the same process if some intermediate library does its own flock.
- **Workaround:** `cross_process_lock=False` (or env var `TINY_QUEUE_NO_FLOCK=1`).
- **Lesson:** always make filesystem-dependent features toggleable. The "zero config" promise is worth less than a knob that lets the user escape a misbehaving kernel call.
- Last verified: 2026-07-03

### 3. regex `^([0-6])W$` won't match `15W`
- The `dom` field accepts things like `15W` (15th-weekday-nearest) but the regex required a single digit `[0-6]` before `W`.
- **Fix:** `^(\d+)W$` to allow multi-digit days (`1W`, `15W`, `31W` are all valid; `31W` is edge).
- **Lesson:** when writing cron-like field regexes, support the full range from the start. Don't assume "weekday" means "0–6".
- Last verified: 2026-07-03

### 4. `os.unlink` on a file with open fd does NOT block on Linux
- I assumed removing a file with an open fd would block. It doesn't (Linux unlinks but keeps the inode alive until last fd closes).
- But the *opposite* — opening a file mid-delete — can block on some filesystems during flock.
- **Lesson:** when in doubt about file/flock semantics, add a knob to disable the cross-process lock and document it.
- Last verified: 2026-07-03

### 5. Multi-format ecosystem cross-linking needs a fallback ladder
- Repos have 4 different ecosystem formats: markdown table (3-col, 2-col), bold bullet list, plain bullet list, lobster-emoji section.
- One regex won't fit them all.
- **Solution:** detect format, then update; fall back to "find end of block and append" for unknown formats.
- Final pass: append a new section to repos with no Ecosystem at all.
- Last verified: 2026-07-03

### 6. Don't accidentally commit embedded git repos from workspace root
- Ran `git add -A && git commit` from the workspace root while working in a subdir → committed nested repos as embedded (warning) and the parent got dirty.
- **Fix:** be paranoid about `pwd` before any git operation that affects the parent.
- **Lesson:** after writing files in a new repo subdir, `cd` there explicitly before any `git` call.
- Last verified: 2026-07-03
